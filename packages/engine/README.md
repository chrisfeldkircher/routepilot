# @routepilot/engine

Framework-agnostic guided tour engine for RoutePilot.

- Tour state machine and registry
- DAG navigation and transitions
- Preparations, demo bridges, and event bridges
- Shared engine types used by framework adapters

Install:

```bash
npm install @routepilot/engine
```

Docs: https://routepilot.dev/docs
Source: https://github.com/chrisfeldkircher/routepilot/tree/main/packages/engine
