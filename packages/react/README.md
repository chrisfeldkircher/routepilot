# @routepilot/react

React bindings for RoutePilot.

- `GuidedTourProvider`
- `GuidedTourOverlay`
- React hooks for state and interactables
- Re-exports from `@routepilot/engine`

Install:

```bash
npm install @routepilot/engine @routepilot/react
```

Docs: https://routepilot.dev/docs
Source: https://github.com/chrisfeldkircher/routepilot/tree/main/packages/react
