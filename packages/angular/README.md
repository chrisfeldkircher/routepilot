# @routepilot/angular

Angular bindings for RoutePilot.

- `GuidedTourService`
- `GuidedTourOverlayComponent`
- Router adapter, directives, and inline-markup pipe
- Re-exports from `@routepilot/engine`

Install:

```bash
npm install @routepilot/engine @routepilot/angular
```

Docs: https://routepilot.dev/docs
Source: https://github.com/chrisfeldkircher/routepilot/tree/main/packages/angular
